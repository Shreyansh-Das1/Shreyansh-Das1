### 🛰️ Flagship Project

**Multi-sensor SAR + Optical Land Cover Classification**
- Fused Sentinel-1 (SAR: VV, VH) + Sentinel-2 (B02, B03, B04, B08) into a 6-channel U-Net pipeline
- **86.57% accuracy · 0.532 mIoU** over ~26M pixels, (EPSG:32645)
- Full preprocessing stack: orbit correction → speckle filtering → terrain correction → reprojection → tiling
- Dataset published on Kaggle · [View on Kaggle](https://www.kaggle.com/shreyanshdas11)
